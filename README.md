# Programação Básica

Este repositório contém exercícios básicos de programação utilizados para prática e aprendizado.

## Objetivo

Praticar conceitos fundamentais de programação, automação e análise de dados utilizando Python.

## Conteúdo

Atualmente o repositório possui exercícios nas seguintes áreas:

- Python básico
- Manipulação de dados
- Automação com PyAutoGUI
- Exercícios de lógica
